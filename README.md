
# MED13 Foundation Website

The official website for the MED13 Foundation, dedicated to supporting families affected by MED13-related disorders.

## Tech Stack

- **Frontend**: React 18 with TypeScript, Vite
- **Backend**: Express.js with TypeScript
- **Database**: PostgreSQL with Drizzle ORM
- **UI Components**: shadcn/ui, Tailwind CSS
- **Routing**: Wouter
- **State Management**: TanStack Query

## Development

```bash
npm install
npm run dev
```

The application will be available at http://localhost:5000

## Deployment

This project is designed to be deployed on Replit with automatic deployments.
